Introduction

Communities stay in touch with PHP at their own pace. This talk focuses on highlighting the features found in PHP 7.0, 7.1 and 7.2. It shows off their benefits and gives you ammo to start upgrading your codebase today.

The talk features the following subjects:

  • PHP 7.0 features
    • return types
    • scalar type hints
    • spaceship operator
  • PHP 7.1 features
    • constant visibility
    • nullables
    • multiple exceptions
  • PHP 7.2 features
    • object type hint
    • abstract function override
    • libsodium
  • ADVISORY: What version should I use today?
  • Code complexity and maintainability
  • WordPress performance on PHP 7
